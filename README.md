# chess-dogrusu

chess dogrusu

2 player chess game written in C. Project took about 1 week to write. I tried to add as many features as I could. Code is not optimized and has some bugs. Check the header file chess.h for details.

There are 2 players. The game starts and each player first types the piece they want to move and then the position for example "E2" then E4". The game continues until a mate occurs.
